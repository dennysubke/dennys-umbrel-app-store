Dumbware Suite

Dumbware is a collection of lightweight self hosted tools for everyday life and organization: simple budgeting, task tracking, file drops, notes, kanban boards, domain lookups and a browser based terminal, all running on your own server.

Apps on this instance

Click any card to open the app in a new tab.

DumbBudget

DumbDo
Clean todo lists for tasks and quick reminders.
DumbDrop
Fast file drop and sharing inside your own network.
DumbPad
Browser based notepad for quick notes and snippets.
DumbKan
Simple kanban boards for projects, workflows and planning.
DumbWhois
Quick WHOIS lookups for domains with a minimal UI.
DumbTerm
Terminal in the browser for fast shell sessions.
DumbAssets
Self hosted asset manager for images, files and media.
