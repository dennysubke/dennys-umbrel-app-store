# 🎮 PokéRogue on Umbrel

### Catch Pokémon, stack absurd items, defeat bosses, and blame critical hits.

---

> [!IMPORTANT]
> PokéRogue runs locally in single player mode without a PokéRogue account.
>
> Your progress is stored in the browser you use to play. It is not automatically synchronized with `pokerogue.net`, other browsers, browser profiles, or devices.

## 🚀 Getting Started

1. Install **PokéRogue**.
2. Open the app from your dashboard.
3. Click once inside the game window so the browser sends keyboard input to the game.
4. Select **New Game**.
5. Choose your starter Pokémon.
6. Begin the run and confidently assume that nothing can possibly go wrong.

No registration, external account, or separate game server is required.

## 🌐 Playing on Another Computer

PokéRogue can be opened from any computer that can access your Umbrel dashboard.

Each browser and browser profile has its own local save data. Opening the game in another browser or on another computer will normally create a separate game state.

For several players in one household, separate browser profiles are the easiest way to keep progress apart.

## 💾 Save Data and Backups

This version uses browser storage because account login is disabled.

Your progress is connected to:

* The computer you are using
* The browser
* The browser profile
* The exact address used to open the app

Changing browsers, switching browser profiles, using a private window, clearing site data, or changing the app address can make the game appear as though no progress exists.

### Create a Manual Backup

Open the game menu with <kbd>Esc</kbd> or <kbd>M</kbd> and look for **Manage Data**.

| Export | Contains |
|---|---|
| **Export Data** | Long term player data such as collection progress and unlocks |
| **Export Session** | The current active run or save slot |

Store exported files somewhere safe before clearing browser data, reinstalling the app, changing the Umbrel address, or moving to another computer.

> [!WARNING]
> Treat this installation as a separate local game. Do not assume that local save data will synchronize with the official online version.

## ⚙️ Recommended Settings

Open the game menu with <kbd>Esc</kbd> or <kbd>M</kbd>, then select **Settings**.

| Setting | Recommendation |
|---|---|
| Tutorials | Enable for the first few runs |
| Game Speed | Begin with 1x or 2x |
| HP Bar Speed | Use Fast once familiar |
| EXP Gain Speed | Use Fast once familiar |
| Fullscreen | Enable when supported |
| Audio | Adjust music and sound effects separately |
| Keyboard | Remap controls to your preference |

Very high game speeds can cause problems on weaker computers. More speed is not always more strategy.

## 🛠️ Troubleshooting

| Problem | What to Try |
|---|---|
| Keyboard controls do not work | Click once inside the game window and try again |
| The browser scrolls instead of controlling the game | Click inside the game and use fullscreen mode |
| There is no sound | Click inside the game and check the browser audio permissions |
| The game feels slow | Enable hardware acceleration in the browser |
| Animations stutter | Reduce the game speed and close unnecessary browser tabs |
| Reward choices appear blank | Wait briefly, reduce the game speed, or reload the page |
| Progress appears missing | Use the same computer, browser, profile, and app address |
| The game behaves strangely after an update | Reload the page or perform a hard refresh |
| A private window forgets progress | Use a normal browser window instead |

> [!CAUTION]
> Clearing cookies, local storage, or all site data can erase local progress. Export your data first whenever possible.

## ❓ Frequently Asked Questions

### Do I need a PokéRogue account?

No. This installation runs as a local single player game without account login.

### Is my progress saved directly on Umbrel?

The game is served by Umbrel, but player progress is stored in the browser. A normal Umbrel app backup should not be treated as a backup of your PokéRogue progress.

### Can I continue the same run on another computer?

Not automatically. Another browser or computer has separate browser storage. Use the available export and import functions where supported.

### Can several people play?

Yes. Each person should use a separate browser profile so that progress does not mix or overwrite another player’s data.

### Why did my perfect strategy fail?

It was probably a critical hit. It is always a critical hit.

## 🧠 Beginner Tips

* Start with a balanced team instead of six favourites with the same weakness.
* Catch Pokémon often to expand your future starter choices.
* Healing items are usually more useful than your pride suggests.
* Held items can stack and create extremely powerful combinations.
* Check enemy information before using the most dramatic attack.
* A failed run still contributes to long term progress.
* One more run is rarely only one more run.
